Introducción a Git

Comandos básicos

  • git init: Inicializa un repositorio nuevo
  • git status: Muestra el estado de los archivos del proyecto
  • git add: Agregar archivos al área de preparado
  • git commit -m "mensaje": Guarda los archivos preparados en el repositorio
  • git log: Muestra el historial de commits que se han hecho
  • git diff: Muestra las diferencias entre archivos del proyecto
  • git checkout: Permite moverse entre el historial de versiones

    • Backtracking

  • git checkout HEAD filename: Descarta cambios en el área de trabajo
  • git reset HEAD filename: Descarta cambios en el área de preparado
